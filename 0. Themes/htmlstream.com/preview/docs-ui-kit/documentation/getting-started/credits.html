Credits

First, let us take this opportunity to thank all the creative minds for their great products and plugins (libraries).

All third-party plugins (libraries) are located in the [assets/vendor/..] folder, while image sources in the [assets/img/..], [assets/img-temp/..] folders and SVG files are in the [assets/svg/..] folder.
